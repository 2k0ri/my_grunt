my grunt template for static pages.

- open and watch `grunt`
	- Jade
	- Sass(+Compass)
	- Stylus
	- CoffeeScript
	- livereload
- compress `grunt dist`
	- csso
	- UglifyJS2

# Usage

``` cmd
> mkdir pages && cd pages
> git clone git://github.com/2k0ri/my_grunt.git .
> bower install
> npm i
> start grunt
```

or

``` sh
$ mkdir pages && cd pages
$ git clone git://github.com/2k0ri/my_grunt.git .
$ bower install
$ npm i
$ grunt &
```
